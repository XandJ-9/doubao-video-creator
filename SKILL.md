# 豆包视频生成技能
调用字节跳动`doubao-seedance-2-0-260128`大模型生成高质量视频，支持文生视频、图生视频两种模式，异步提交任务无需等待，自动保存任务记录，生成完成自动下载结果。
## 触发规则
当用户提到以下内容时，**必须**触发本技能：
- 生成视频、创建视频、视频制作、短视频生成
- 豆包视频、seedance视频、豆包动画、AI视频生成
- 文生视频、图生视频、图片转视频、动漫视频制作
- 查询视频生成任务、下载生成好的视频
## 功能特性
1. **支持三种生成模式**
   - ✅ 文生视频：纯文本描述生成视频，支持中文精准理解
   - ✅ 单图生视频：上传一张图片作为首帧，生成连贯动态视频
   - ✅ 首尾帧过渡：上传两张图片，生成两者之间的自然过渡动画
2. **自定义参数**
   - 时长支持5-15秒
   - 分辨率比例支持16:9（横屏）、9:16（竖屏）、1:1（方形）、adaptive（自适应图片比例）
   - 支持生成带环境音效的有声视频
   - 可开关水印
3. **异步任务管理**
   - 提交任务立即返回，无需等待生成过程
   - 自动保存所有任务记录到本地，包含提示词、提交时间、状态、保存路径
   - 支持批量查询所有任务状态，生成完成自动下载视频
4. **原生支持中文**
   - 对中文提示词语义理解准确率高，生成符合中文用户审美需求的视频
   - 支持动漫、写实、3D、手绘等多种风格描述
## 使用流程
### 步骤1：确认用户需求
- 视频内容描述：主题、风格、画面细节、运镜方式、动作描述等，越详细生成效果越好
- 视频时长：5-15秒，默认5秒
- 视频比例：默认16:9横屏，可选择9:16竖屏、1:1方形
- 是否需要生成音效：默认不需要
- 是否需要水印：默认添加
- 是否有参考图片：如果有，支持图生视频模式
### 步骤2：提交生成任务
执行`doubao-video submit`命令，按照要求传入参数：
```bash
# 文生视频
doubao-video submit --prompt "动漫风格，蓝天下风吹草地，阳光明媚，镜头缓慢拉近" --duration 5 --ratio 16:9
# 图生视频
doubao-video submit --prompt "人物微笑眨眼，头发被风吹动，背景流动" --image ./reference.jpg --duration 3 --ratio adaptive
```
提交成功后返回任务ID，告知用户后续可以查询状态。
### 步骤3：查询任务状态
执行`doubao-video query`命令查询所有进行中的任务，或者指定任务ID查询：
```bash
# 查询所有进行中的任务
doubao-video query
# 查询指定任务
doubao-video query cgt-xxxx
```
任务生成完成会自动下载到`output/`目录，返回保存路径给用户。
### 步骤4：手动下载（可选）
如果需要重新下载，可以执行：
```bash
doubao-video download cgt-xxxx --output ./my_videos --filename result.mp4
```
## 最佳实践
### 提示词优化
- 风格描述：明确指定风格，例如"新海诚风格"、"吉卜力手绘风格"、"赛博朋克风格"、"国风水墨风格"
- 画面细节：描述光线、色彩、场景、人物穿着动作等，例如"傍晚日式街道，橙红色夕阳，少年穿黑橙相间羽织手持武士刀"
- 运镜方式：例如"镜头缓慢拉近"、"360度环绕运镜"、"平移镜头"、"特写镜头"
- 动态描述：例如"风吹动草叶和发丝"、"花瓣飘落"、"刀身反射寒光"
### 版权规避
请勿在提示词中使用受版权保护的IP名称（如"鬼灭之刃"、"迪士尼"、"漫威"等），改用风格和特征描述，例如将"鬼灭之刃炭治郎"改为"日式热血动漫风格少年，穿黑橙相间羽织，耳戴圆形日轮耳饰，手持武士刀"。

## 使用示例
### 示例1：生成动漫风格视频
**用户输入**：帮我生成一个5秒的动漫视频，内容是蓝天下风吹过油菜花田，阳光明媚

### 示例2：根据图片生成视频
**用户输入**：帮我用这张人物头像生成一个3秒视频，内容是人物眨眼睛微笑，头发飘动
**技能执行**：
```bash
doubao-video submit --prompt "年轻女孩微笑眨眼，黑色长发被风吹动，背景虚化流动" --image ./avatar.jpg --duration 3 --ratio 1:1
```
### 示例3：查询任务状态

**技能执行**：
```bash
doubao-video query
```
**返回结果**：

> 描述：吉卜力风格，春日蓝天下大片油菜花田...
> 状态：succeeded

## 常见问题处理
### 生成失败提示"InvalidCredential"
API密钥无效，请检查config.yaml中填写的API Key是否正确。
### 生成失败提示"QuotaExceeded"
账号额度不足，请充值后再试。
### 生成失败提示"版权限制"
提示词中包含受版权保护的内容，请修改提示词，改用风格和特征描述，不要直接使用IP名称。
### 生成时间过长
10秒视频通常需要3-5分钟，属于正常情况，提交任务后可以继续其他工作，后续查询即可。
